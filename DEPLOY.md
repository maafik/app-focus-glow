# Развертывание на GitHub Pages

## Что нужно сделать:

### 1. Включить GitHub Pages

1. Перейдите в репозиторий → Settings → Pages
2. В разделе "Build and deployment" выберите:
   - Source: **GitHub Actions**
3. Нажмите "Save"

### 2. Настроить Vite для GitHub Pages

Нужно добавить базовый путь в `vite.config.ts`:

```typescript
export default defineConfig(({ mode }) => ({
  base: '/app-focus-glow/', // Замените на имя вашего репозитория
  // ... остальная конфигурация
}));
```

### 3. Процесс деплоя

После настройки:
1. Внесите изменения в код
2. Сделайте commit и push в основную ветку
3. GitHub Actions автоматически соберет и опубликует проект

### 4. Адрес сайта

Ваш сайт будет доступен по адресу:
`https://[username].github.io/app-focus-glow`

### 5. Если нужен кастомный домен

1. В Settings → Pages добавьте свой домен
2. Настройте DNS записи:
   - A: `185.199.108.153`, `185.199.109.153`, `185.199.110.153`, `185.199.111.153`
   - CNAME: `username.github.io` (если нужно)

### 6. Преимущества GitHub Pages

- ✅ Бесплатно
- ✅ Автоматический деплой
- ✅ HTTPS по умолчанию
- ✅ CDN от GitHub
- ✅ Поддержка кастомных доменов

## Если возникнут проблемы

Проверьте:
- GitHub Actions включен в настройках репозитория
- Права доступа для Actions (Settings → Actions → General)
- Базовый путь в vite.config.ts соответствует имени репозитория
